# TODONotes

web-сервис для управления TODO-заметками.

REST API на backend и React на frontend.

Сохранение todo для различных проектов и личный кабинет для разных типов пользователей.

## Стек

- Python > 3.7
  - isort, black, autoflake
  - Django < 3.3
  - DRF
- REACT
- JS
- VSCode
- SQLite 3
